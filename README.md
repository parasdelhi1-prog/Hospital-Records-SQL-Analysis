## 📖 Project Overview

I recently completed an SQL assignment provided by **Satish Dhawale** (founder of Skill Course), where I analyzed realistic hospital records to extract meaningful business insights. This project simulated real-world healthcare scenarios, demonstrating how organizations rely on data-driven decisions.

---

## 🎯 Dataset Overview

The analysis covered a hospital dataset with:
- **Hospital names** and locations
- **Departments** (General Practice, Cardiology, Neurology, etc.)
- **Patient counts** and doctor availability
- **Medical expenses** tracking
- **Admission/discharge dates** for stay duration analysis

---

## 💼 Business Questions Answered

Through 10 practical SQL queries, I answered:

| # | Question | Insight |
|---|----------|---------|
| 1 | Total patients across hospitals? | 9,216+ patients |
| 2 | Average doctors per hospital? | Resource planning |
| 3 | Top 3 busiest departments? | Capacity planning |
| 4 | Hospital with max expenses? | Cost analysis |
| 5 | Daily average expenses? | Budget tracking |
| 6 | Longest hospital stay? | Patient care duration |
| 7 | Patients per city? | Geographic analysis |
| 8 | Average stay per department? | Operational efficiency |
| 9 | Least busy department? | Resource optimization |
| 10 | Monthly expense trends? | Financial reporting |

## 🛠️ SQL Techniques Used

**Core Functions:**
- `SUM()`, `AVG()` – Aggregation
- `GROUP BY` – Data categorization
- `ORDER BY` – Sorting results
- `LIMIT` – Top/bottom rankings
- `Date_Trunc()`, Date subtraction – Time-based analysis

**Concepts Applied:**
- Business requirement translation
- Data aggregation and reporting
- Time-series analysis
- Performance comparison
